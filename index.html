Three positives

  • Free
  • Debugger for Chrome: It allows you to set breakpoints and debug JS from within VSCode
  • Fast due to it's built on Node.js,Electron, HTML and CSS --> searching and no lags with opening and changing files. (Apparently on par with Sublime which is a paid editor)
  • Options for customization

Three negatives
